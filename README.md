\# Blackjack Game



A simple command-line Blackjack game built using Python.



\## Features



\* User vs Computer gameplay

\* Blackjack detection

\* Ace value adjustment (11 -> 1)

\* Score comparison and winner determination



\## Technologies Used



\* Python

\* Random module



\## How to Run



python main.py
